# Somali Dataset Repository System 
#The Somali Dataset Repository is a dual-platform system with a web portal and mobile app designed to collect, organize, and share Somali datasets. It supports dataset submission, metadata management, and secure access for researchers, government, and NGOs.

#Development Environment:

#•Backend: Node.js, Express
•Database: MongoDB Atlas (cloud)

#•Frontend (Web): React
•Mobile: Flutter
